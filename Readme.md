# AI Agent Workshop

The repository contains Google Cloud Run Day Workshop materials with 4 distinct workshop modules demonstrating progressive AI agent development patterns using Google's Agent Development Kit(ADK).

Key insights from the analysis:

- Workshop 1: Basic agent with news retrieval functionality
- Workshop 2: Demonstrates Google Cloud Storage integration via MCP servers
- Workshop 3: Agent-to-Agent communication with orchestrator pattern
- Workshop 4: Advanced features with LiteLLM and NLP capabilities
